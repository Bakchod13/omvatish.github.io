# Om Vatish Portfolio Template (GitHub Pages)

This is a simple **IT + Photography + Streaming** portfolio template designed for GitHub Pages.
No build tools. Just upload files and it works.

## Quick Start (GitHub Pages)
1. Put these files into your repo named: `omvatish.github.io`
2. Go to **Settings → Pages**
3. **Source:** Deploy from branch → `main` → `/ (root)`
4. Visit your site:
   - `https://YOURUSERNAME.github.io/` (if repo is `YOURUSERNAME.github.io`)
   - or `https://YOURUSERNAME.github.io/REPO/` (if different)

## What to edit
- `index.html`: your main content
- `blog.html` + `/blog/*.html`: blog posts (optional)
- `assets/css/style.css`: design
- `assets/js/main.js`: animations + lightbox
- Replace images inside `assets/img/`
- Put your resume in: `assets/Om-Vatish-Resume.pdf`

## Tip
Keep the site simple for recruiters:
- 3 IT projects
- 9 best photos
- 1 showreel link
- 1 blog post (optional)

Good luck 💪
